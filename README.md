# DevSheets Website

Ultra-modern marketing website for DevSheets built with Astro, React, Tailwind CSS, and Framer Motion.

## Features

- ⚡ Lightning-fast static site generation with Astro
- 🎨 Smooth animations with Framer Motion
- 📱 Fully responsive design
- 🎭 Modern gradient effects and glassmorphism
- 🚀 Optimized for Netlify deployment

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is configured for automatic deployment on Netlify. Simply:

1. Push to GitHub
2. Connect your repo to Netlify
3. Netlify will auto-detect Astro and deploy

Or use the Netlify CLI:

```bash
netlify deploy --prod
```
