## 🌐 Deploy to Netlify

### Quick Deploy
```bash
cd website
npm run build
netlify deploy --prod
```

### Or via GitHub
1. Push to GitHub
2. Go to https://app.netlify.com
3. "Add new site" → "Import from Git"
4. Select your repo
5. Netlify auto-detects Astro settings
6. Deploy!

## 🎯 What Makes This Site Fast

1. **Astro**: Static generation = near-zero JavaScript
2. **Lazy Loading**: Components load only when visible
3. **Optimized Fonts**: Preloaded with display=swap
4. **SVG Icons**: No image downloads needed
5. **Tailwind Purging**: Only used CSS shipped
6. **Netlify CDN**: Global edge caching
7. **Aggressive Caching**: 1-year cache for static assets
